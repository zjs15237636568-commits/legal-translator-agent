## 验收标准（MVP）

1. 上传一份 ≤20MB 的涉外支付协议（docx/pdf），10 分钟内完成中英双栏呈现
2. 术语库 CRUD 可用，新增术语能在新翻译中生效
3. 翻译完成后能生成红/黄风险卡片 ≥5 条，点击可跳转原文段落
4. Q&A 回答带 `[Clause X.X]` 引用，点击能跳转并高亮
5. 断网/APIKey 失效后可恢复，失败段可单独重试
6. 能导出中英对照 Word 文件，含风险批注
7. 支持在 OpenAI / Claude / DeepSeek 三者间切换，无需改代码
